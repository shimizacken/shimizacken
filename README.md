**Bouvet - Sesam VS Code Extension** (Mar 2026 - present)

A full-featured VS Code extension for the Sesam integration platform built as a TypeScript monorepo (`sesam-ts`) with three packages: `@sesam/core`, `@sesam/cli`, and the VS Code extension itself. Architected using Language Server Protocol (LSP).

Key features delivered:
- DTL language support: syntax highlighting for ~160 built-in DTL functions, autocomplete with signatures and documentation, hover documentation, real-time linting with quick fixes
- Code navigation: go-to-definition, find all references, rename rules, cross-file navigation
- Configuration completion: intelligent suggestions for pipe, system, and source configurations
- Node integration: upload, download, and run pipes directly from VS Code; live real-time updates via Socket.IO
- Sync status and diff view: git-like visualization of differences between local configs and the Sesam node
- Credential management: secure storage via VS Code SecretStorage API with multi-environment support
- Test management: VS Code Testing API integration with green/red inline results
- GitHub Copilot `@sesam-agent` chat participant

**Bouvet - Vibe Coding in the Frontend** (Apr 2026 - present)

Designed and developed a 4-day AI-assisted development workshop for frontend developers using GitHub Copilot as the primary tool. 11 modules covering the full development lifecycle - from planning and scaffolding to implementation, testing, and debugging. Includes dedicated modules on security (OWASP Top 10, secrets handling, trust boundaries), GitHub Actions CI/CD, and agentic workflows.
